Despre GIT și GitHub

GIT este un sistem de control al versiunilor creat de Linus Torvalds în 2005, utilizat pentru a urmări modificările aduse fișierelor dintr-un proiect. Este un instrument indispensabil pentru echipele de dezvoltatori care lucrează la același cod.

GitHub este o platformă bazată pe cloud care folosește GIT pentru gestionarea proiectelor. Ea permite colaborarea, partajarea codului și găzduirea site-urilor web prin GitHub Pages.

Avantajele utilizării GitHub

  • Acces rapid la codul sursă de oriunde
  • Posibilitatea de a lucra simultan pe mai multe ramuri (branches)
  • Transparență și control total al versiunilor
  • Integrare cu instrumente de automatizare (CI/CD, Actions, Copilot)
